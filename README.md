# 🕹️ Проект Unity: Программирование Игрока и Паркур

## 📜 Обзор проекта
В этом проекте демонстрируется, как создать простую систему передвижения персонажа, используя Unity и C#, с дополнительным творческим заданием по разработке курса паркура. Проект продвигается от внедрения базовых элементов управления персонажем к разработке камеры с видом от первого лица, систем поддержания здоровья и выносливости.

## 🎯 Задачи
1. Реализовать основные движения персонажа с помощью Unity и C#.
2. Запрограммировать механику прыжков для игрового персонажа.
3. Создать поле для паркура, по крайней мере, с 5 объектами, по которым игрок сможет пройти.
4. Установить камеру с видом от первого лица и разработать систему поддержания здоровья и выносливости.
5. Расширить паркур, включив в него как минимум **5 дополнительных платформ**, которые будут проходить с видом от первого лица.

---

## 🏗️ Творческое задание: Разработать паркурную трассу

### 1. 🧱 Создать препятствия для паркура
Использовать **кубы** и другие формы, чтобы построить как минимум **5 платформ**, между которыми игрок должен прыгать.

### 2. 🎯 Расширяем возможности паркура
Разработать дополнительные **5 платформ** с общим количеством **10 препятствий**, чтобы обеспечить прохождение трассы с учетом способностей игрока.
